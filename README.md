
# colomain's dotfiles

## Requirements

Set zsh as your login shell:

```
> chsh -s $(which zsh)
```

## Install

### Clone onto your laptop:

```
> git clone git://github.com/colomain/dotfiles.git ~/dotfiles
```

Install [stow](https://www.gnu.org/s/stow/):

		
### Install the dotfiles:

```
> cd ~/dotfiles
~/dotfiles> stow .
```
